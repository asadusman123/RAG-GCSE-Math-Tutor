# Medium Questions

1 The interior angle of a regular polygon with n sides is 150°.
Calculate the value of n.
n = ................................................

Answer

Method 1
It is a regular polygon so all the interior angles are equal and all the exterior angles are equal.
An interior angle and an exterior angle add up to 180°. Use this to find the size of an exterior angle.
150° + Exterior angle = 180°
Exterior angle = 180° - 150°
Exterior angle = 30°
The formula for the exterior angle of a regular n-sided polygon is Exterior angle = 360 / n. Use this to form an equation.
30 = 360 / n
[1]
Multiply both sides by n.
30n = 360
Divide both sides by 30.
n = 360 / 30
